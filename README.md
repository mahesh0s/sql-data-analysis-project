# SQL Data Analysis Project

This project demonstrates SQL skills used for data analysis and reporting in real-time business scenarios.

## Skills Used
- SQL Queries
- Joins
- Group By & Aggregate Functions
- Subqueries
- Database Relationships (Primary & Foreign Keys)

## Project Description
Created relational database tables such as Employees, Departments, Customers, and Orders.  
Wrote advanced SQL queries to generate business insights like:

- Department-wise salary report  
- Highest & second highest salary  
- Customer total spending  
- Monthly sales report  
- Employee and department analysis

SQL (MySQL/Oracle/SQL Server)
